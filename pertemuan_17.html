Tag div sering digunakan untuk mengelompokkan beberapa elemen menjadi satu bagian agar lebih mudah diberi gaya dengan CSS atau dimanipulasi menggunakan JavaScript. Elemen ini juga dapat diberi atribut seperti class, id, atau title untuk keperluan styling dan pengaturan tertentu
Padding digunakan untuk membuat jarak antara konten dan border elemen. Kita dapat mengatur padding untuk setiap sisi elemen, baik ke atas atau bawah, maupun kanan atau kiri
Margin dalam CSS memungkinkan web developer mengatur tampilan dan layout elemen di sebuah halaman web. Dengan fungsi utamanya sebagai ruang kosong di sekitar elemen, margin berperan penting dalam menciptakan keseimbangan, struktur, dan estetika visual situs.
